Priority Modes Documentation

Priority modes define the logic by which the system combines setpoints for the photovoltaic plant (PV) and the battery energy storage system (BESS), in order to control the overall power flow to or from the electrical grid.

  • PV power setpoint ≥ 0 → PV export target
  • BESS power setpoint > 0 → BESS is discharging
  • BESS power setpoint < 0 → BESS is charging
